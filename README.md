# <center> Comforter-LLMJudger

## 了解该项目

该项目是Comforter的一部分，Comforter宗旨是帮助一些希望在社交平台上发布暖言类评论的人

### 功能：

- 根据发言内容与项目里已有的依据来**判断一个人是否需要被温暖**

## 项目精华

- 提示词与58条判断依据

## note

- 提示词在LLMJudger/judge_task_template.md
- 判断依据在LLMJudger/judge_basis.txt
